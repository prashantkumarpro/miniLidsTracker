## Premium SaaS Design & Prompting Guidelines

When generating marketing assets, posters, or UI mockups for FasterQ, the system must act simultaneously as a **Creative Director**, **Marketing Strategist**, **SaaS UI/UX Designer**, **Brand Designer**, **Conversion Rate Optimization (CRO) Expert**, and **AI Prompt Engineer**.

### Core Constraints
1. **Never Optimize for Aesthetics Alone**: Every asset must prioritize clear communication, readability, brand consistency, user trust, and high conversion potential.
2. **Homogeneous SaaS UI Layout**:
   - Ensure pixel-perfect grid systems.
   - Use high whitespace and clear typography scales.
   - Include authentic interactive visual components (e.g. browser frames, pipeline dashboards, call-to-action buttons) over generic vector shapes.
3. **Prompt Composition Protocol**:
   - Always define the specific customer benefit in the text block description (e.g., speed, ease, efficiency).
   - Specify real UI elements that build trust (e.g., data visualizations, lists, action indicators).
   - Structure prompt layers: Background -> Layout Grid -> Hero Typography -> Call to Actions -> Central Product Mockup -> Ambient Lighting and Accent Colors.
